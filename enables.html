Understanding Enables

In the BMS the enable is generally just a please run signal to the equipment. Most run hour is just how long has it been enabled for.

  • To get true run time you need a DPS (differential pressure switch) or flow meter/sensor however these are sometime not installed due to cost.
  • Some have current switches, but these just prove that power is being drawn. If the pump or fan is broken it can be drawing power but not moving.
  • Example: BMS is saying that a pump is enabled but it is off on the panel. The enabled signal will be on in the controller regardless. So you would need to look at some sort DPS/Flow or current switch for the purpose of Virtual Metering.
